# **App Name**: Black Ticker

## Core Features:

- Cinematic Hero Intro: Dramatic page load sequence revealing logo, headline, and 3D floating tickets with a ticket-shaped mask animation.
- Interactive Event Cards: Display a grid of 3D rotating, tiltable event cards with glassmorphism UI, responding to cursor hover and scroll.
- Animated QR Ticket Scanner: Showcase a digital ticket UI with an animated QR code scan, laser effect, and verification sequence.
- Horizontal Scroll Gallery: A scroll-triggered horizontal event gallery with cards scaling dynamically based on their position in view.
- Dashboard Feature Explosion: Animated presentation of platform dashboard mockups where UI components 'explode' outwards with scale, slide, and glow effects.
- Parallax Event Environment: Create an immersive live event simulation with multiple parallax layers for crowd, lights, and fog, animated stage lighting, and a text overlay.
- Scroll-Driven Booking Timeline: Visualize the booking process through a timeline where steps, drawing icons, and progress lines animate upon scroll.

## Style Guidelines:

- A subtle, dark black and red combo only for a sophisticated, futuristic base (#15171A).
- black for main calls to action and highlights, evoking modern technology ().
- red to complement the blue, adding a luxurious, energetic glow ().
- Headlines: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, bold feel.
- Body text: 'Inter' (sans-serif) for clean readability and a contemporary aesthetic. Note: currently only Google Fonts are supported.
- Utilize 'Lucide Icons' for crisp, vector-based iconography, supporting the clean and futuristic UI.
- Implement an immersive scroll experience with advanced scroll animations, horizontal sections, and integrated 3D objects, achieving a cinematic feel.
- Adopt glassmorphism, neon glow, soft shadows, and animated gradients for UI elements, enhanced by subtle particle backgrounds and parallax depth layers.
- All animations are scroll-driven, smooth, and cinematic, using GPU-accelerated transforms for optimal 60fps performance without layout shifts. Animations combine translate, scale, rotate, opacity, and blur.
- Integrate a custom glowing cursor with magnetic button effects, a glow trail, and slight scaling on card hovers, adding an extra layer of interactivity.
