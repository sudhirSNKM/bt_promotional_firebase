
# Black Ticker

Black Ticker is a cinematic event ticketing platform designed for the extraordinary. It leverages cutting-edge technology to provide a high-end, immersive user experience for elite event management and attendance.

## Features

- **Cinematic Experience**: A visually stunning landing page with 3D floating elements and scroll-driven animations.
- **Immersive Gallery**: Horizontal scroll sections for showcasing featured events.
- **Secure Verification**: Proprietary QR technology for seamless and secure event access.
- **Advanced Dashboard**: Full control for event organizers with real-time analytics and management tools.
- **Engaging UI**: Custom cursor effects, parallax backgrounds, and success animations.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS, ShadCN UI
- **Icons**: Lucide React
- **AI Integration**: Genkit (Google Gemini)
- **Backend**: Firebase Firestore & Authentication
